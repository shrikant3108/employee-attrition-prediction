# employee-attrition-prediction

## 📌 Problem Statement
Predict whether an employee will leave the company.

## ⚙️ Algorithms Used
- Logistic Regression
- L1 Regularization
- L2 Regularization

## 📊 Results
- Test Accuracy: 87.03%
- Mean CV Accuracy: 87.11%
- Mean F1 Score: 86.99%

## 🛠 Technologies
Python, Pandas, NumPy, Scikit-learn
